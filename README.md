# E-Commerce-Store-Demo
Single-file e-commerce store demo with real product images, built in vanilla HTML/CSS/JS.
A fully functional dark-theme e-commerce demo built with zero frameworks or dependencies. Features a product listing page with category filters, full product detail pages with image galleries, and a working cart drawer with quantity management — all in one HTML file.
Sections: Product Grid · Category Filters · Product Detail · Image Thumbnails · Cart Drawer · Toast Notifications
Stack: HTML5 · CSS3 · Vanilla JS · Unsplash (product images)
Features: Add to cart · Qty controls · Running total · Sale/New badges · Color & size selectors · Fully responsive
